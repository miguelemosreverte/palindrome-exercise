Architecture

Railway Container ($5/mo, ~4GB RAM) ├── proxy.js (port 8080, sticky routing) ├── OpenCode #0 (port 9001, /workspaces/pool-0/) ├── OpenCode #1 (port 9002, /workspaces/pool-1/) └── OpenCode #2 (port 9003, /workspaces/pool-2/) Each instance: own filesystem, own DB, own sessions. Proxy: hash-based sticky routing, health checks, auto-restart.
